# SEO: Google Search Console and sitemap

After deploying the site, register it with Google so search engines can discover and index your pages.

## 1. Add the property in Google Search Console

1. Open [Google Search Console](https://search.google.com/search-console).
2. Click **Add property**.
3. Choose **URL prefix** and enter: `https://www.vimutti.ru`
4. Complete ownership verification (HTML file upload, DNS record, or HTML meta tag as suggested by GSC).

## 2. Submit the sitemap

1. In Search Console, open your property.
2. Go to **Sitemaps** in the left sidebar.
3. In **Add a new sitemap**, enter: `sitemap.xml`
4. Click **Submit**.

The sitemap is generated by the app at `https://www.vimutti.ru/sitemap.xml` and includes the home page, magazine index, all category listing pages, and all article URLs.

## 3. Optional: request indexing for important URLs

In **URL Inspection**, you can request indexing for specific pages (e.g. the homepage or key articles). Indexing is not instant; it may take from a few days to a few weeks depending on crawl budget.

## 4. Yandex (for Russian audience)

For [Yandex Webmaster](https://webmaster.yandex.ru/), add the site and submit the same sitemap URL: `https://www.vimutti.ru/sitemap.xml`.
